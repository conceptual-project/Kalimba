# 🎵 Калимба — Интерактивная веб-калимба

Мобильное веб-приложение (PWA) — калимба с реалистичным звуком. Играйте мелодии прямо в браузере, 17 клавиш с синтезированным звучанием калимбы и эффектом реверберации.

Работает **полностью офлайн**, устанавливается на домашний экран как обычное приложение.

![PWA](https://img.shields.io/badge/PWA-ready-green?style=flat-square) ![License](https://img.shields.io/badge/license-MIT-blue?style=flat-square)

## ✨ Возможности

- 🎹 **17 клавиш** — стандартная калимба в до мажоре (C4–E6)
- 🔊 **Реалистичный звук** — синтез через Web Audio API с обертонами и реверберацией
- 🎨 **Красивый интерфейс** — деревянный корпус, металлические клавиши, резонаторное отверстие
- 📳 **Виброотклик** — тактильная обратная связь при нажатии
- 🎼 **Лента нот** — отображение последних сыгранных нот
- 🏷️ **Режимы подписей** — ноты, номера или оба варианта
- ⌨️ **Клавиатура** — играйте клавишами 1-9, 0, Q, W, E, R, T, Y, U
- 📴 **Офлайн** — работает без интернета благодаря Service Worker
- 🏠 **Установка** — полноценное PWA

## 🐛 Исправленные баги

При аудит исходного кода были найдены и исправлены следующие баги:

### 1. 🔴 3-я гармоника не подключена к реверберации (Критический)
**Проблема:** В функции `playKalimbaNote()` 3-я гармоника (`osc3` / `g3`) была подключена только к `ctx._masterDry`, но не к `ctx._masterReverb`. Все остальные осцилляторы (основной тон, 2-я гармоника, хор) подключались к обоим каналам, что создавало непоследовательное пространственное звучание — 3-я гармоника звучала «сухо».

**Исправление:** Добавлено подключение `g3.connect(ctx._masterReverb)` для единообразной пространственной обработки всех гармоник.

### 2. 🟠 Импульсный шум (body resonance) не подключён к реверберации (Средний)
**Проблема:** Шумовой импульс body resonance (фильтрованный шум, имитирующий резонанс корпуса) также подключался только к `ctx._masterDry`, но не к `ctx._masterReverb`. Это приводило к тому, что атака звука калимбы звучала «плоско» без пространственной глубины.

**Исправление:** Добавлено подключение `gn.connect(ctx._masterReverb)` для естественной реверберации атаки.

### 3. 🟠 Состояние гонки в анимации резонаторного отверстия (Средний)
**Проблема:** Функция `triggerRipple()` переключала CSS-класс `animate` через `remove()` / принудительный reflow / `add()` на одном и том же элементе `#ripple`. При быстром повторном вызове (например, во время глиссандо) класс удалялся и добавлялся в одном кадре, и браузер часто не перезапускал CSS-анимацию — визуальный «кольцевой» эффект вокруг резонаторного отверстия пропускался.

**Исправление:** Заменён подход на **key-based remount** — каждый вызов `triggerRipple()` удаляет старый ripple-элемент и создаёт новый с классом `animate`. Это гарантирует перезапуск анимации при каждом нажатии. Статический элемент `#ripple` из HTML удалён за ненадобностью.

### 4. 🟡 Отсутствие обработки ошибок AudioContext (Низкий)
**Проблема:** При создании `AudioContext` в `getCtx()` не обрабатывались ошибки. На некоторых браузерах/устройствах создание контекста может завершиться неудачей (например, из-за политик автоплея или отсутствия поддержки), и приложение выбросило бы необработанное исключение.

**Исправление:** Обёрнуто в `try/catch` с `console.warn` и возвратом `null`. Функция `playKalimbaNote()` проверяет результат и делает ранний `return` — визуальная обратная связь продолжает работать даже без звука (graceful degradation).

### 5. 🟡 Отсутствие `aria-pressed` и `aria-keyshortcuts` на клавишах (Низкий)
**Проблема:** Кнопки клавиш калимбы имели только `aria-label`, что снижало доступность для экранных читалок — пользователи не могли понять, нажата ли клавиша в данный момент, и не знали горячих клавиш.

**Исправление:**
- Добавлен атрибут `aria-pressed="false"` при создании каждой клавиши, который динамически переключается в `"true"` / `"false"` в `playNote()`.
- Добавлен `aria-keyshortcuts` (например, `"1"`, `"Q"`), чтобы скринридеры озвучивали горячую клавишу для каждой клавиши.

### 6. 🟡 Порядок объявления `KEY_MAP` (Temporal Dead Zone) (Низкий)
**Проблема:** После добавления `aria-keyshortcuts` объект `KEY_MAP` стал использоваться в цикле построения клавиш (раньше использовался только в обработчике `keydown`). Но `KEY_MAP` объявлялся через `const` ниже по коду — это приводило бы к `ReferenceError` из-за временной мёртвой зоны (TDZ) при первом рендере.

**Исправление:** `KEY_MAP` перенесён в секцию данных (после `NOTE_COLORS`) и объявляется один раз. Дублирующее объявление удалено.

### 7. 🟡 Ошибка форматирования в таблице README (Низкий)
**Проблема:** Последняя строка таблицы цветовой кодировки имела лишний символ `|` после «Сиреневый», что нарушало форматирование Markdown-таблицы.

**Исправление:** Удалён лишний разделитель.

## 🎹 Расположение клавиш

Стандартная 17-клавишная калимба в до мажоре. Клавиши расположены как на настоящей калимбе — самая длинная (C4) в центре, укорачиваются к краям:

Цвет:   🟠   🟣   🔵   🟡   🔴   🔵   🟢   🟠   🔴   🟡   🔵   🟣   🟠   🟢   🔵   🔴   🟡

### Левая и правая рука

На настоящей калимбе левый большой палец играет клавиши 1–8 (левая сторона), а правый — клавиши 10–17 (правая сторона). Клавиша 9 (C4) в центре доступна обоим пальцам.

**Левая сторона (клавиши 1–8):** D6, B5, G5, E5, C5, A4, F4, D4
**Центр (клавиша 9):** C4
**Правая сторона (клавиши 10–17):** E4, G4, B4, D5, F5, A5, C6, E6

## 🎼 Как играть мелодии

Ниже — популярные мелодии, записанные двумя способами:
1. **Нотами** — названия нот (C4, D4, E4...)

### 11. 🔥 Rammstein — Du Hast (клавишный проигрыш)

Знаменитый синтезаторный рифф из конца песни «Du Hast» — один из самых узнаваемых в тяжёлой музыке! Песня звучит в ми миноре, но рифф идеально ложится на калимбу в до мажоре — все ноты натуральные, без диезов и бемолей.

**Рифф повторяется циклично.** Каждая группа — один такт. Играйте ровно, с чётким ритмом (четвертными нотами), как марш. Акцент на первую ноту каждой группы.

```
Основной паттерн (повторять):

Ноты:   B4 B4 B4 B4 | C5 C5 C5 C5 | B4 B4 B4 B4 | C5 B4 A4 B4
Номера: 12 12 12 12 | 5  5  5  5  | 12 12 12 12 | 5  12 6  12
        ───────────   ───────────   ───────────   ─────────────
        лев.рука     прав.рука     лев.рука      обе руки
```

**Советы:**
- Клавиша 12 (B4) — левая сторона, клавиша 5 (C5) — правая сторона. Используйте оба больших пальца!
- Играйте строго в ритме: ТА-та-та-та | ТА-та-та-та
- Последняя группа (C5 B4 A4 B4) — развязка фразы, тут мелодия спускается и возвращается
- Для эффекта Rammstein играйте жёстко, с акцентом

### 12. 🌊 Nightwish — Nemo (клавишный проигрыш в начале)

Атмосферная клавишная тема из вступления к «Nemo» — визитная карточка Nightwish! Оригинал звучит в до-диез миноре, но для калимбы в до мажоре мы транспонируем в ля минор — звучит так же красиво, и все ноты есть на инструменте.

**Интро играет до начала вокала.** Нежно, плавно, с эхо-эффектом (реверберация калимбы сама создаст нужную атмосферу).

```

Ноты:   E5 D5 C5 D5 | E5 — E5 D5 | C5 D5 E5 D5 | C5 — A4 —
Номера: 4  13 5  13 | 4  — 4  13 | 5  13 4  13 | 5  — 6  —
        ───────────   ──────────   ────────────   ──────────
        прав.рука                  обе руки


Фраза 2 (продолжение):

Ноты:   E5 D5 C5 D5 | E5 E5 D5 C5 | D5 C5 A4 C5 | D5 — — —
Номера: 4  13 5  13 | 4  4  13 5  | 13 5  6  5  | 13 — — —
        ───────────   ───────────   ────────────   ────────
        прав.рука                   обе руки
```

**Советы:**
- Играйте мягко, с лёгким «протягиванием» каждой ноты — пусть звук плавно затухает
- Клавиши 4 (E5) и 13 (D5) — правая сторона; клавиша 5 (C5) — правая сторона; клавиша 6 (A4) — левая
- Реверберация калимбы создаст атмосферу симфонического металла
- Фраза 1 и фраза 2 чередуются — играйте их по очереди
- Старайтесь к мягкому пиано — Nightwish в интро звучит воздушно

## 💡 Советы для начинающих

1. **Начните с простого** — Twinkle Twinkle Little Star идеальна для первой попытки
2. **Используйте номера** — переключите режим подписей на «Номера», чтобы легче было следовать табулатурам
3. **Слушайте ритм** — играйте ровно, не торопитесь
4. **Два пальца** — как на настоящей калимбе, используйте большие пальцы обеих рук
5. **Практика** — играйте мелодию медленно, пока не запомните, затем ускоряйте
6. **Следите за лентой нот** — ноты внизу экрана помогают отслеживать, что вы сыграли
7. **Для рок-мелодий** — Rammstein и Nightwish звучат на калимбе неожиданно красиво! Контраст тяжёлого оригинала и нежного тембра калимбы создаёт особую магию

## 🚀 Установка

### Как PWA на телефон

**Android (Chrome):**
1. Откройте сайт в Chrome
2. Нажмите ⋮ → **«Установить приложение»**

**iOS (Safari):**
1. Откройте сайт в Safari
2. Нажмите ⊹ → **«На домашний экран»**

### Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите содержимое этой папки в корень репозитория
3. Включите GitHub Pages: **Settings → Pages → Branch: main**
4. Готово! Приложение доступно по адресу `https://ваш-логин.github.io/репозиторий/`

### Локальный запуск

Это статическое PWA — никакого сервера не нужно. Просто откройте `index.html` в браузере, либо запустите любой статический сервер:

```bash
# Python 3
python3 -m http.server 8000

# или Node.js
npx serve .

# или Bun
bunx serve .
```

Затем откройте `http://localhost:8000` в браузере.

## 🛠 Технологии

- **Web Audio API** — синтез звука калимбы с обертонами и реверберацией
- **ConvolverNode** — реалистичный эффект реверберации
- **Service Worker** — офлайн-работа через Cache-first стратегию
- **Web App Manifest** — установка на рабочий стол, standalone-режим
- **Vibration API** — тактильная обратная связь
- **Touch Events** — поддержка мульти-тач для игры двумя пальцами
- **CSS Animations** — визуальная обратная связь при игре
- **ARIA** — доступность (`aria-label`, `aria-pressed`, `aria-keyshortcuts`)

## 🎶 О калимбе

Калимба (калимпы, mbira) — африканский ударный инструмент, также известный как «thumb piano» (фортепиано для больших пальцев). Звук извлекается ударом больших пальцев по металлическим язычкам. Калимба имеет мягкий, тёплый, колокольчиковый тембр, похожий на музыкальную шкатулку.

Стандартная калимба в до мажоре охватывает диапазон от C4 (до малой октавы) до E6 (ми третьей октавы) — 17 клавиш, которых достаточно для исполнения множества популярных мелодий.
